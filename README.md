<h1>Ticket Manager</h1>
<p>A simple ASP.NET REST API for managing a ticket system, with frontend using Blazor.</p>
<h2>v0.0.1 (Stable)</h2>
<p>This is a backup of the version in which the weather page on the Blazor app template was able to pull data from the weatherforcast method in the API template using localhost.</p>
<p>This was achieved using Cross-Origin Resource Sharing (CORS) to allow the Blazor frontend to take in data from the API, despite it being hosted on a different localhost port</p>
